# Excel Pack Specification

Workbook name: `{company}_pack.xlsx`

## Sheets
1. Summary
   - A1: Company
   - B1: Company name
   - A2: Latest Revenue
   - B2: Latest revenue value
   - A3: Latest Runway (months)
   - B3: Runway value
2. P&L Actuals
   - Full KPI table by period
3. KPI Trends
   - period, revenue, gross_margin, ebitda, runway_months
4. Forecast
   - Forecasted KPI table for next 12 months
5. Scenarios
   - Scenario parameters
6. Scenario Comparison
   - Ending revenue, EBITDA, and runway by scenario
7. Scenario Deltas
   - Delta vs Base for revenue, EBITDA, and runway
8. Sensitivity
   - 2D driver grid results
9. Notes
   - Generation metadata
